Создай веб-приложение CardMaker на Replit. Стек: React (фронтенд), Node.js + Express (бекенд), PostgreSQL, Supabase Auth.

**Партнёрская программа (простая):**
- Новый пользователь получает 3 бесплатные генерации.
- При регистрации можно указать реферальный код пригласившего (параметр ?ref=КОД).
- За каждого приведённого друга партнёр получает +3 бонусные генерации.
- Награда начисляется сразу после регистрации друга.
- Бонусные генерации можно использовать для создания карточек.

**База данных:**
- users: id, email, name, auth_id, referral_code (уникальный), referrer_id (ссылка на партнёра), free_generations (по умолч. 3), bonus_generations (по умолч. 0), created_at
- referrals: id, referrer_id, referred_id, created_at
- bonus_transactions: id, user_id, amount (всегда +3), source, referral_id, created_at
- generations: id, user_id, input_image_url, output_text, output_image_url, created_at

**Логика:**
1. При регистрации:
   - Генерация referral_code (например, USER-{id}-{random}).
   - Если передан ref=КОД → найти реферера, установить referrer_id, создать запись в referrals.
   - Новому пользователю free_generations = 3.
   - Если есть реферер → bonus_generations реферера +3, запись в bonus_transactions.
2. При создании карточки:
   - Списывать сначала bonus_generations, затем free_generations.
   - Если баланс 0 → показать сообщение о необходимости пополнения (пока заглушка).
3. В личном кабинете:
   - Раздел "Партнёрская программа": код, ссылка, статистика (сколько привёл, сколько бонусов получено), история начислений.

**Интерфейс:**
- Главная: заголовок "Карточка товара за минуту", кнопки "Создать карточку", "Войти".
- Страница генерации: загрузка фото, поле цены, выбор маркетплейса, результат (текст + изображение).
- Личный кабинет: история, баланс (free / bonus), раздел партнёрской программы.
- Регистрация/вход через Supabase Auth, поле реферального кода (опционально), поддержка ?ref=КОД в URL.

**AI интеграции:**
- OpenAI Vision (gpt-4o) для анализа фото и генерации JSON описания.
- KIE AI (Nano Banana) для создания финального изображения карточки с инфографикой.

Все ключи в Secrets.
